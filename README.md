## 📌 Overview

This project is a **Java Swing Based Multi-Client Chat Application** that allows multiple users to communicate in real-time using **Socket Programming**.

It follows a **Client-Server Architecture** where:
- The server handles all client connections
- Clients can send and receive messages instantly
- Multiple users can chat simultaneously

## ⚙️ How To Run

### 1️⃣ Compile Files

```bash
javac ChatServer.java
javac ChatClient.java
```

---

### 2️⃣ Start Server

```bash
java ChatServer
```

---

### 3️⃣ Run Client

```bash
java ChatClient
```

You can open multiple clients to simulate multiple users chatting simultaneously 💬
