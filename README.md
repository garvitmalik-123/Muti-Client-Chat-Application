# 🚀 How To Run

## 1️⃣ Clone Repository

```bash
git clone https://github.com/garvitmalik-123/your-repo-name.git
cd your-repo-name
```

---

## 2️⃣ Compile Files

```bash
javac project_2/*.java
```

---

## 3️⃣ Start Server

```bash
java project_2.ChatServer
```

---

## 4️⃣ Start Client

```bash
java project_2.ChatClient
```

---

# 🌐 LAN Usage

If users are connected on the same WiFi/LAN:

1. Run the server on one machine.
2. Copy the server IP.
3. Enter that IP while starting client.
4. Start chatting in real-time.
